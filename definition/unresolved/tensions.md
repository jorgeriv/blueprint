# Tensions

Status: active

## Current Tensions

- Blueprint should be structured enough to guide agents, but not so rigid that
  it crushes early creative thinking.
- The CLI should be useful immediately, but the deeper value will come from
  skills that do not exist yet.
- Placeholder files make structure visible, but they should not be mistaken for
  finished project definition.
- Blueprint should be inspired by Spec Kit without becoming software-only or
  implementation-first.
