# Problem

Status: active

Many tools assume a project is already clear enough to manage. They help with
tasks, issues, specs, or implementation plans, but they do not focus on the
earlier phase where a person is still discovering what the project actually is.

In that early phase, important reasoning is often scattered across chat,
brainstorming, intuition, and partial notes. Without a structure for distilling
that material, future work can lose the reasons behind decisions or rush into
implementation too soon.

Blueprint addresses this gap by helping humans and AI agents turn early
conversation into a navigable project definition.
