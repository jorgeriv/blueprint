# Audience

Status: active

Blueprint is for people who are shaping a new idea and want an AI agent to help
clarify it before execution.

Likely users include:

- software builders defining a new project before implementation
- creators shaping a creative, business, research, or personal project
- teams that want to preserve the reasoning behind early decisions
- AI-assisted workflows that need stable context before agents begin building

The first user is the Blueprint project itself, which is using Blueprint to
define and refine its own structure.
